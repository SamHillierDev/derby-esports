# Derby Esports

---

Derby Esports is a professional competitive esports organisation and gaming community located in the United Kingdom.

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v22.13.1
```

```bash
npm -v
# 11.1.0
```

## Installation

Run the commands:

```bash
npm install
```

```bash
npm run dev
```
